# Task10- (Animation to Hero Image)

## Project Description
A responsive Laundry Service Landing Page built using HTML, CSS, and Flexbox, enhanced with a custom CSS animation for the hero section image.

This project focuses on creating an engaging UI by adding orbit movement, pause, and squeeze effects using pure CSS (no JavaScript).

## Features
- Responsive Navigation Bar
- Flexbox-based layout
- Hero section with text + image
- Book a Service button 
- Mobile responsive design
- Custom CSS animation (no JS)

## Animation Details
The hero image includes a custom animation with the following sequence:

- Animation Flow
- Image moves in an orbit path using translate
- Comes back to original position
- Pauses briefly
- Applies squeeze effect using scale()
- Returns to normal
- Repeats infinitely

## Key Concept Used
CSS Transform
transform: translate(x, y);
transform: scale(x, y);

**translate()** → moves the image in different directions

**scale()** → creates squeeze/stretch effect
